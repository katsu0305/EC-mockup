# 実装手順書 Phase 6 配布対応

## 1. 目的

GUI アプリを PyInstaller で実行ファイル化し、非開発者へ配布可能にする。

## 2. 開始条件

- Phase 5 が完了していること
- GUI 版の基本動作が確認済みであること

## 3. 実施手順

### 3-1. 配布前提を整理する

1. 同梱対象ファイルを決める
2. 非同梱ファイルを決める
3. .env 配置ルールを決める
4. 作業ディレクトリの配置ルールを決める

### 3-2. PyInstaller 設定を作る

1. spec ファイルを作る
2. GUI エントリポイントを指定する
3. 必要な data files を同梱する
4. ログ出力先を確認する

### 3-3. 実行ファイルを生成する

1. ビルドコマンドを整備する
2. 配布物の構成を確認する
3. 初回起動確認を行う

### 3-4. 配布検証を行う

1. 開発環境外で起動確認する
2. .env 読み込み確認をする
3. build 実行確認をする
4. 出力先の作成確認をする

### 3-5. 配布手順書を作る

1. インストール不要かを明記する
2. .env 配置方法を記載する
3. 実行方法を記載する
4. よくある失敗を記載する

## 4. 成果物

- 配布可能な GUI 実行ファイル
- PyInstaller spec
- 配布手順書

## 5. 完了判定

- 実行ファイル単体で GUI が起動する
- .env を読んで build が実行できる
- 配布手順書だけで第三者が操作できる

## 6. 注意点

- .env や secret を配布物へ同梱しない
- 開発環境依存のパスを埋め込まない
- 同梱漏れを必ず別環境で検証する
